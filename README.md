# Olympic Medals Project

This project looks into whether wealthier countries, larger countries, or host countries win more Olympic medals.

## Data sources
- Kaggle Olympic medals dataset
- World Bank GDP data
- World Bank population data
- World Bank surface area data

## Team tasks
- Data engineering pipeline
- Analysis and modeling
- Visualization and reporting
