# Olympic Medals Data Engineering Project

## Research Question
Do wealthier countries (GDP), larger countries (population / size), or host countries win more Olympic medals?

## How to Run — Step by Step

### Step 1 — Install dependencies
Install all required Python packages:
```bash
pip install -r requirements.txt
```

### Step 2 — Run the data pipeline
This single command runs the entire pipeline in order:
```bash
python src/run_pipeline.py
```

What it does internally:
1. **World Bank API** — downloads GDP, population, and surface area data for all countries
2. **Clean Olympics** — filters the raw dataset to Summer Olympics medal winners only, aggregates to country level per year
3. **Merge datasets** — joins Olympic results with World Bank data, adds host country flag, creates derived features (log GDP, GDP per capita, etc.), saves to SQLite database
4. **Spark processing** — uses Apache Spark to calculate total medals distributed per Olympic year

After this step, `data/processed/` will contain all generated files.

### Step 3 — Open the analysis notebook
Open `notebooks/olympic_analysis.ipynb` in Jupyter or VS Code and click **Restart & Run All**.

The notebook runs in order:
1. **Top 10 countries** — SQL query on the SQLite database showing most successful nations
2. **Medal inflation** — line chart from Spark output showing how total medals have grown over time
3. **Correlation heatmap** — shows how strongly GDP and population correlate with medal counts
4. **Host country effect** — bar chart comparing average medals when hosting vs not hosting
5. **OLS Regression** — statistical model quantifying how much GDP, population, surface area, and hosting independently affect medal counts
6. **Mann-Whitney U test** — statistically tests whether the host country advantage is significant
7. **Random Forest** — machine learning model that predicts medal counts and ranks which factors matter most

Charts are automatically saved to `charts/` as PNG files.

### Step 4 — Launch the interactive dashboard
Start the Streamlit web dashboard:
```bash
streamlit run dashboard/app.py
```

Then open your browser at: **http://localhost:8501**

The dashboard lets you:
- Filter by year range
- Change how many top countries are shown
- Set a minimum medal threshold
- See all 4 charts update live based on your filters

## Data Sources

| Dataset        | Source                            | What it contains                           |
|----------------|-----------------------------------|--------------------------------------------|
| Olympic medals | Kaggle                            | Every Olympic medal winner since 1896      |
| GDP            | World Bank API (`NY.GDP.MKTP.CD`) | Total economic output per country per year |
| Population     | World Bank API (`SP.POP.TOTL`)    | Total population per country per year      |
| Surface area   | World Bank API (`AG.SRF.TOTL.K2`) | Country size in km²                        |

---

## Key Engineering Challenges & Solutions

**1. Country code mismatch**
Olympic data uses NOC codes (e.g. `GER`, `SUI`) while World Bank uses ISO3 codes (`DEU`, `CHE`). A manual mapping layer translates between the two, including historical nations like the Soviet Union (`URS` → `RUS`).

**2. Missing data before 1960**
World Bank economic data is sparse before 1960. The dataset is restricted to 1960 onwards for reliability.

**3. Messy team names**
The raw data contains clubs, mixed teams, and historical entities. The pipeline filters on structured 3-letter country codes rather than team names to avoid this.

**4. Multi-source integration**
Four separate datasets with different schemas are standardised and merged on (country code, year) as the shared key.

## Features Created by the Pipeline

| Feature              | Description                                                 |
|----------------------|-------------------------------------------------------------|
| `medals_total`       | Gold + silver + bronze medals per country per year          |
| `log_gdp`            | Natural log of GDP (reduces skew for modeling)              |
| `log_population`     | Natural log of population                                   |
| `gdp_per_capita`     | GDP divided by population                                   |
| `medals_per_million` | Medals per 1 million people                                 |
| `host_flag`          | 1 if the country hosted the Olympics that year, 0 otherwise |
